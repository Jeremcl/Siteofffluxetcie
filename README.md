# Flux & Cie - Site Web

Site web moderne avec animations Framer Motion et design futuriste.

## Installation

```bash
npm install
```

## Développement

```bash
npm run dev
```

## Build

```bash
npm run build
```

## Prévisualisation

```bash
npm run preview
```

## Technologies

- React 18
- Vite
- Framer Motion
- Tailwind CSS
- Lucide React (Icônes)
